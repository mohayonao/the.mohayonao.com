# [6chars drums](http://the.mohayonao.com/6chars/)

> The Rhythm machine that is composed with units of 6 digits of hexadecimal.

```
aa0881 aa08a0 (WALK THIS WAY)
HH: x-x-x-x- x-x-x-x- (aa aa)
SD: ----x--- ----x--- (08 08)
BD: x------x x-x----- (81 a0)
```
